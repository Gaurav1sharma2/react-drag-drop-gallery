# React Drag & Drop Gallery

A React project demonstrating login functionality with drag-and-drop image gallery interactions.

## Features

- **Login Page**: Simple authentication with dummy credentials
- **Drag & Drop Gallery**: Reorder images by dragging and dropping
- **Image Gallery**: Display images in a responsive grid
- **Visual Feedback**: Hover and dragging effects for better UX
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Installation

```bash
npm install
```

## Running the Project

```bash
npm start
```

The app will open at `http://localhost:3000`

## How to Use

1. Login with the demo credentials
2. Drag any image to reorder the gallery
3. Drop the image in the desired position
4. The gallery will update automatically

## Technologies Used

- React 18
- React Router v6
- CSS3 (Drag & Drop API)
- HTML5 Drag and Drop API
